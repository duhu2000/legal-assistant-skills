# 交互式简历集成 - Dify 指南

> 扩展功能：为网页简历添加聊天窗口，让面试官可以通过对话了解你的作品

---

## 功能概述

通过在网页简历中嵌入 Dify 聊天组件，你可以：
- 让面试官通过对话了解你的作品、文章、研究
- 展示你的技术能力（AI 集成）
- 提供更丰富的信息展示方式
- 形成差异化竞争优势

---

## 前置要求

### 必需
- Dify.ai 账号（免费版即可）
- Claude Code（用于生成和修改 HTML）
- 你的作品/文章内容（作为知识库）

### 可选
- Dify API Key（使用第三方模型）
- 网页部署平台（Vercel、GitHub Pages 等）

---

## 操作步骤

### 第一步：在 Dify 中创建应用

1. 登录 [Dify.ai](https://cloud.dify.ai)
2. 进入「工作室」
3. 点击「创建应用」
4. 选择「聊天助手」

### 第二步：配置应用

#### 2.1 基础信息
- **应用名称**：如「{姓名}的作品助手」
- **应用描述**：如「关于{姓名}的文章、研究、项目」

#### 2.2 提示词编写

```
你是一位专业的助手，负责向面试官介绍{姓名}的作品、文章和研究。

你的职责：
1. 介绍{姓名}的专业背景和工作经历
2. 详细说明{姓名}的文章、研究内容
3. 回答关于{姓名}的作品的具体问题
4. 展示{姓名}的专业能力和思考方式

你的回答风格：
- 专业、准确、简洁
- 基于知识库中的事实
- 不夸大、不虚构
- 主动引导面试官了解{姓名}的优势

注意事项：
- 只回答关于{姓名}及其作品的问题
- 如果不知道答案，诚实说明
- 不涉及隐私信息
```

#### 2.3 配置知识库

1. 点击「添加知识库」
2. 上传你的作品：
   - 文章（Markdown/PDF）
   - 研究报告
   - 项目说明
   - 论文摘要
3. 选择合适的文本分割方式
4. 保存知识库

#### 2.4 配置模型（可选）

如果你有第三方 API Key：
- 点击「模型提供商」
- 添加你的 API（如 DeepSeek）
- 选择模型（如 deepseek-chat）
- 调试确保可用

### 第三步：发布应用

1. 点击「发布」
2. 在「访问方式」中选择「嵌入网站」
3. 复制嵌入代码

### 第四步：集成到简历

1. 将 Dify 嵌入代码发送给 Claude Code
2. 请求将其添加到简历 HTML 中
3. 调整位置和样式

#### 提示词示例

```
请将以下 Dify 聊天组件代码添加到我的简历 HTML 中：

[粘贴 Dify 嵌入代码]

要求：
1. 将聊天窗口固定在页面右下角
2. 初始状态为收起，显示一个按钮
3. 点击按钮展开聊天窗口
4. 样式与简历整体风格保持一致
5. 添加"作品助手"标签
```

## 本地测试

### 问题
直接用浏览器打开 HTML 文件时，Dify 组件可能无法使用（CSP 限制）

### 解决方案
使用 VS Code 的 Live Server 扩展：

1. 安装 VS Code
2. 安装「Live Server」扩展
3. 右键点击简历 HTML
4. 选择「Open with Live Server」
5. 在浏览器中测试聊天功能

---

## 部署到网络

### 选项 1：Vercel（推荐）

```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 在简历目录运行
vercel

# 3. 按提示操作
# 4. 获得部署 URL
```

### 选项 2：GitHub Pages

1. 创建 GitHub 仓库
2. 上传简历 HTML
3. 在仓库设置中启用 GitHub Pages
4. 选择主分支作为源
5. 获得 URL：`https://{username}.github.io/{repo}/resume.html`

### 选项 3：Netlify Drop

1. 访问 [app.netlify.com/drop](https://app.netlify.com/drop)
2. 拖放简历 HTML 文件
3. 等待部署完成
4. 获得部署 URL

## 常见问题

### Q1: Dify 组件不显示

**可能原因**：
- CSP 策略限制
- 网络问题
- API 配置错误

**解决方案**：
- 使用 Live Server 本地测试
- 检查 Dify 应用是否正确发布
- 检查 API Key 是否有效

### Q2: 聊天回复不准确

**可能原因**：
- 知识库内容不足
- 提示词不够清晰
- 模型能力限制

**解决方案**：
- 补充更多作品到知识库
- 优化提示词
- 尝试更强大的模型

### Q3: 担心隐私泄露

**建议**：
- 知识库中只包含公开信息
- 不在知识库中放入敏感内容
- 设置 Dify 应用的访问限制

## 进阶优化

### 1. 自定义样式

```css
/* 修改 Dify 组件样式 */
.dify-chat-container {
    bottom: 20px;
    right: 20px;
    width: 350px;
    max-height: 500px;
}

.dify-chat-button {
    background: linear-gradient(135deg, #D4AF37 0%, #C9A227 100%);
    border-radius: 50%;
    width: 60px;
    height: 60px;
}
```

### 2. 添加欢迎消息

在 Dify 应用设置中添加开场白：
```
你好！👋 我是{姓名}的作品助手。

你可以问我：
• {姓名}有哪些专业文章？
• 关于{某文章}的核心观点是什么？
• {姓名}在{某领域}有什么研究？
• {姓名}的项目经验如何？

点击下方问题快速开始 👇
```

### 3. 添加快捷问题

在聊天窗口添加预设问题按钮，方便面试官快速了解：
```
[按钮] 介绍{姓名}的专业背景
[按钮] {姓名}有哪些主要文章？
[按钮] 关于{某领域}{姓名}有什么研究？
```
