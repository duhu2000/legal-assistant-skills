# MCP 降级查询模板

> 每个 MCP 工具对应的 Web Search 降级查询模板

---

## 降级查询策略

### 原则
1. **多轮搜索**：每个 MCP 对应 2-3 个不同角度的查询
2. **精准匹配**：使用引号确保公司名精确匹配
3. **来源验证**：优先选择官方来源（法院网站、政府网站）

---

## 搜索技巧

### 1. 公司名称变体

```
如果公司全称搜索无结果，尝试：
- 简称：如"腾讯"而非"腾讯科技（深圳）有限公司"
- 英文名：如"Tencent"
- 股票简称：如"腾讯控股"
- 曾用名：如公司曾更名
```

### 2. 时间限定

```
使用时间限定获取最新信息：
- "{公司名}" 被执行 2024
- "{公司名}" 失信 近一年
```

### 3. 精确匹配

```
使用引号确保精确匹配：
- "{公司全称}" 而非 {公司全称}
- "{公司名}" AND "被执行" 使用 AND 连接关键词
```

### 4. 来源限定

```
限定搜索来源：
- "{公司名}" site:zxgk.court.gov.cn （执行信息网）
- "{公司名}" site:wenshu.court.gov.cn （裁判文书网）
```
