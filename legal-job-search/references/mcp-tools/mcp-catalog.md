# 支持的 MCP 工具目录

> 本目录说明法律AI求职助手支持的 MCP 功能及其降级查询方案
> **注意**：不限定具体 MCP 工具名称，支持任何提供相同功能的 MCP 工具

---

## 企业法律风险类

### 功能匹配原则

本 Skill 采用**功能优先**的策略，支持任何提供以下功能的 MCP 工具：

```
匹配依据：
│
├─ 1. 工具名称中包含相关关键词
│   ├─ 中文：被执行、失信、限制高消费、终本、裁判文书等
│   └─ 英文：executed, dishonest, consumption, final, judgment 等
│
├─ 2. 工具描述中包含相关功能说明
│   └─ 如："查询企业被执行信息"、"返回失信被执行人记录"等
│
└─ 3. 工具参数包含企业名称相关字段
    ├─ entname
    ├─ company_name
    └─ 其他合理的公司名称参数
```

---

### 1. 企业被执行信息

**功能需求**：查询企业被执行人信息、案件数量、执行标的等

**MCP 功能匹配**：
- 任何提供企业名称参数，能返回被执行信息的 MCP 工具
- 工具名称可能包含：`executed`、`execution`、`enforcement`、`被执行`、`执行` 等关键词

**典型参数**：
- `entname`（企业全称）
- `company_name`（公司名称）

**降级查询**：`"{公司名}" 被执行人信息`
**替代网站**：中国执行信息公开网、企查查、天眼查

**风险等级评估**：
- 无被执行案件：无风险
- 1-3起案件，标的较小：低风险
- 3-5起案件或标的较大：中风险
- 5起以上或标的巨大：高风险

## 使用建议

1. **优先使用 MCP**：MCP 工具返回的数据更精确、更全面
2. **降级到 Web Search**：MCP 不可用时，使用降级查询并明确告知用户数据来源
3. **数据验证**：Web Search 获取的数据建议用户提供二次验证
4. **扩展集成**：用户如果配置了其他法律相关 MCP，可以动态添加到分析流程中
