## Web Search 搜索策略

### 搜索轮次

```
轮次1：律所基础信息
├─ 搜索："{律所名}" + "官网" + "律所介绍"
├─ 搜索："{律所名}" + "基本信息" + "律师人数"
└─ 整合：成立时间、规模、分所

轮次2：业务领域
├─ 搜索："{律所名}" + "业务领域" + "专业领域"
├─ 搜索："{律所名}" + "擅长业务" + "经典案例"
└─ 整合：核心业务、业务特点

轮次3：行业排名
├─ 搜索："{律所名}" + "钱伯斯" + "排名"
├─ 搜索："{律所名}" + "ALB" + "Legal500"
└─ 整合：行业排名、荣誉奖项

轮次4：团队信息
├─ 搜索："{律所名}" + "合伙人" + "团队"
├─ 搜索："{律所名}" + "律师" + "招聘"
└─ 整合：团队结构、招聘需求

轮次5：薪酬水平（如可查）
├─ 搜索："{律所名}" + "薪酬" + "待遇"
├─ 搜索：脉脉/知乎 + "{律所名}" + "薪资"
└─ 整合：薪酬范围、福利待遇
```
