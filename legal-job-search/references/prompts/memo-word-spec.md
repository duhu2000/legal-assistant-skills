# 面试备忘录 Word 文档规范

> 输出格式：精美的 .docx 文档
> 适用场景：面试前快速查阅的专业备忘录

---

## 文档样式规范

### 页面设置

```yaml
page:
  size: A4
  margin:
    top: 2.5cm
    bottom: 2.5cm
    left: 2cm
    right: 2cm
```

### 字体规范

```yaml
fonts:
  title:
    family: "微软雅黑" / "PingFang SC"
    size: 18pt
    bold: true
    color: "#1C3F5E"

  section_title:
    family: "微软雅黑" / "PingFang SC"
    size: 14pt
    bold: true
    color: "#2C5282"

  subsection_title:
    family: "微软雅黑" / "PingFang SC"
    size: 12pt
    bold: true
    color: "#5A6C7D"

  body:
    family: "宋体" / "SimSun" / "STSong"
    size: 10.5pt
    color: "#2C3E50"

  table_header:
    family: "微软雅黑" / "PingFang SC"
    size: 10pt
    bold: true
    color: "#FFFFFF"

  table_body:
    family: "宋体" / "SimSun"
    size: 10pt
    color: "#2C3E50"
```

### 配色方案

```yaml
colors:
  # 主色 - 用于标题和强调
  primary:
    main: "#D4AF37"      # 金色
    dark: "#C9A227"      # 深金色

  # 辅色 - 用于章节标题
  secondary:
    main: "#1C3F5E"      # 深蓝色
    light: "#2C5282"     # 中蓝色

  # 表格配色
  table:
    header_bg: "#1C3F5E"  # 表头背景
    header_text: "#FFFFFF" # 表头文字
    border: "#E0E0E0"     # 边框颜色
    stripe: "#F8F9FA"     # 斑马纹颜色
```

## 生成工具使用

使用 `docx` skill 生成 Word 文档：

```python
# 使用 docx skill 的关键命令

from docx import Document
from docx.shared import Pt, Inches, RGBColor
from docx.enum.text import WD_ALIGN_PARAGRAPH

# 创建文档
doc = Document()

# 设置页面
section = doc.sections[0]
section.page_height = Inches(11.69)  # A4 高度
section.page_width = Inches(8.27)    # A4 宽度
section.top_margin = Inches(1)       # 2.54cm
section.bottom_margin = Inches(1)
section.left_margin = Inches(0.8)    # 2cm
section.right_margin = Inches(0.8)

# 添加标题
title = doc.add_paragraph('面试备忘录')
title.alignment = WD_ALIGN_PARAGRAPH.CENTER
title_run = title.runs[0]
title_run.font.size = Pt(18)
title_run.font.bold = True
title_run.font.color.rgb = RGBColor(0x1C, 0x3F, 0x5E)

# 添加表格
table = doc.add_table(rows=3, cols=4)
table.style = 'Light Grid Accent 1'

# 设置表头
header_cells = table.rows[0].cells
for cell in header_cells:
    cell.text = '标题'
    cell.paragraphs[0].runs[0].font.color.rgb = RGBColor(0xFF, 0xFF, 0xFF)
    cell.background.color = RGBColor(0x1C, 0x3F, 0x5E)

# 保存文档
doc.save('面试备忘录_XX公司.docx')
```
