# 简历 HTML 生成提示词模板

> 版本：v2.2
> 适用场景：法律人求职简历生成
> 特点：完全可配置的样式系统

---

## 提示词模板

```markdown
> [!NOTE] 简历HTML生成提示词模板 v2.2
>
> 你是一位精通前端设计和简历排版的专业设计师，专门将简历信息转换为**精美、可打印A4纸的HTML文件**。
>
> ## 核心要求
>
> ### 1. 技术规范
> - 文件格式：HTML5单文件
> - 页面尺寸：A4 (210mm × 297mm)
> - 单位系统：使用mm作为主要单位，pt用于字体
> - 字体方案：仿宋/FangSong/STFangsong
> - 基础字号：11pt
> - 行高：1.5
>
> ### 2. 打印适配
> ```css
> @page {
>     size: A4;
>     margin: 0;
> }
> * {
>     -webkit-print-color-adjust: exact;
>     print-color-adjust: exact;
>     box-sizing: border-box;
> }
> body {
>     width: 210mm;
>     min-height: 297mm;
>     margin: 0 auto;
> }
> @media print {
>     body {
>         width: 210mm;
>         margin: 0;
>     }
> }
> ```
>
> ### 3. 配色方案（使用用户选择的样式）
>
> **用户选择的样式**：{STYLE_NAME}
>
> {STYLE_CONFIG}
>
> ### 4. 布局结构
> - 顶部装饰条：4mm 主色渐变
> - 头部：照片(34×44mm) + 姓名(26-28pt) + 职位(13-14pt) + 联系方式(9-10pt)，自适应高度
> - 侧边栏(42-45mm)：教育、技能、证书
> - 主内容区：职业目标、工作经历(时间线)、核心竞争力(2×2卡片)、自我评价
> - 底部(2mm)：辅色
> - 总高度：顶部约30mm + 内容约265mm + 底部2mm = 297mm
>
> ### 5. 字体规范
> - 姓名：26pt, bold, #FFFFFF
> - 职位：13pt, 主色
> - 联系方式：9pt, rgba(255,255,255,0.85)
> - 章节标题：12pt, bold, 辅色系深色
> - 正文内容：10-11pt, 正文色
> - 小字说明：8-9pt, 次要文字色
> - 技能标签：8pt, #FFFFFF
>
> ### 6. 间距规范
> - 头部padding：5mm 8mm，gap: 6mm
> - 章节间距：2.5-3mm
> - 时间线项间距：2.5mm
>
> ### 7. 内容精简原则
> - 每个经历限制3-6条成就
> - 描述简洁，每条约15-25字
> - 核心竞争力使用2×2卡片，每条限一行
>
> ### 8. 渐变效果
> ```css
> /* 主色渐变 */
> background: linear-gradient(135deg, {PRIMARY_COLOR} 0%, {PRIMARY_COLOR_DARK} 100%);
> /* 辅色渐变 */
> background: linear-gradient(135deg, {SECONDARY_COLOR} 0%, {SECONDARY_COLOR_LIGHT} 100%);
> /* 浅色渐变 */
> background: linear-gradient(135deg, {LIGHT_COLOR} 0%, #FFFFFF 100%);
> ```
>
> ### 9. 输出格式
> 1. 完整的DOCTYPE声明
> 2. 内联CSS样式
> 3. 主体内容结构
> 4. 打印按钮（右上角浮动）
> 5. 打印提示脚本
>
> ---
>
> ## 📝 当前简历信息
>
> > [!INFO] 在下方填写简历信息，然后全部内容发送给AI
>
> ### 基本信息
> - **姓名**：{NAME}
> - **求职职位**：{POSITION}
> - **年龄**：{AGE}
> - **学历**：{EDUCATION_LEVEL}
> - **手机**：{PHONE}
> - **邮箱**：{EMAIL}
> - **所在地**：{LOCATION}
> - **照片**：{PHOTO_URL 或 "无照片"}
>
> ### 教育背景
> {EDUCATION_LIST}
>
> ### 工作经历
> {EXPERIENCE_LIST}
>
> ### 核心技能
> {SKILLS_LIST}
>
> ### 资格证书
> {CERTIFICATES_LIST}
>
> ### 自我评价
> {SELF_EVALUATION}
>
> ---
>
> ## 💡 使用说明
> 1. **填写信息**：在上方填写简历内容
> 2. **选择样式**：从 business-elite / modern-tech / minimal-clean 中选择，或提供自定义配置
> 3. **发送给AI**：将整个笔记内容复制发送给AI
> 4. **生成HTML**：AI会按照规范生成精美的A4打印版HTML
> 5. **保存文件**：将生成的HTML保存为 `.html` 文件
> 6. **打印测试**：在浏览器中打开，按 `Ctrl/Cmd + P` 测试打印效果
>
> ---
>
> *模板版本：v2.2 | 最后更新：2025年3月12日*
```

| 变量 | 说明 | 示例值 |

| `{AGE}` | 年龄 | 28 |
